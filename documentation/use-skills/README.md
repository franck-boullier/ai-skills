# Documentation - How To Use Skills

Skills were initially invented by Anthropic (who gave us the Claude family of LLM and tools like Claude Code and Claude Cowork).

They have become a de-facto standard for creating and sharing specific knowledge, processes, and methods that AI agents can use.

Skills are now supported by several tools, including:

- [Cursor - the AI first IDE](xxx)
- [Claude Code](xxx)
- [Claude Cowork](xxx)
- [Microsof Copilot - Cowork](xxx)
- [Github Copilot](xxx)
- [Google Antigravity](xxx)
- [Google Gemini CLI](xxx)
- [Amazon Kiro](xxx)
- [Microsoft VS Code](xxx)
- [Open AI Codex](xxx)

This is a non-exhaustive list, and more and more Agentic AI tools are supporting the Skills format.

What does that mean? It means that you can create a skill using the Skills format and then use it in ANY TOOL that supports this format.

You can even use a AI Agent Orchestration Tool like Claude Cowork to help you create a skill, export that skill, and then use that skill in another AI Agent Orchestration Tool like Cursor, VS Code, or any other tool that supports the Skills format.
