# Generic Skills

Skills that are not specific to any domain or use case, but can be used in a variety of contexts. These skills are designed to be flexible and adaptable, allowing them to be used in a wide range of applications.

## Skills

- [Read PDF Document](./read-pdf-document/SKILL.md): A skill that reads the content of a PDF document and extracts it as text.
- [Text to Markdown](.//txt-to-markdown/SKILL.md): A skill that converts plain text into well-formatted markdown.
