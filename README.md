# AI Skills

Skills files for Agentic AI

[Agent skills](https://agentskills.io/home) is an open format that allows you to create specific skills that AI agents can use.

Several tools are supporting the Skills format, including:

- [Anthropic Claude](https://code.claude.com/docs/en/skills).
- [Cursor - an AI natvie Integrated Development Environment (IDE)](https://cursor.com/docs/context/skills).
- [Google Antigravity](https://antigravity.google/docs/skills).
- [Amazon Kiro](https://kiro.dev/docs/skills/).
- [Microsoft VS Code](https://code.visualstudio.com/docs/copilot/customization/agent-skills).

Skill files are following a [standard format](https://agentskills.io/what-are-skills) which makes them Portable (you can create a skill and use it in ANY TOOL that supports the Skills format).

This repository is a collection of skills that I have created for my own use.
