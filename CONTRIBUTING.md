# Contributing to AI Skills

Thank you for your interest in contributing to the **AI Skills** repository! We welcome contributions that help expand the collection of high-quality, portable skills for AI agents.

## How to Contribute

### 1. Adding New Skills

If you have a skill that follows the [standard format](https://agentskills.io/what-are-skills), we would love to have it!

- **Location**: Place your skill in a new subdirectory under [skills/](./skills/).
- **Structure**: Each skill should have its own folder containing a `SKILL.md` file and any necessary supporting scripts or references.
- **Documentation**: Ensure your `SKILL.md` clearly explains what the skill does and how to use it.

### 2. Improving Documentation

Found a typo? Want to add a guide for a new AI tool?

- Documentation files are located in the [documentation/](./documentation/) directory.
- Feel free to submit PRs for clarity, formatting, or new tool integrations.

### 3. Bug Reports & Suggestions

- Open an [Issue](https://github.com/franckboullier/ai-skills/issues) if you find a problem or have an idea for a new feature.

## Development Workflow

1. **Fork** the repository.
2. **Clone** your fork locally.
3. **Create a branch** for your changes: `git checkout -b feature/my-new-skill`.
4. **Commit** your changes with clear messages.
5. **Push** to your fork and **open a Pull Request**.

## Code of Conduct

Please be respectful and collaborative in all interactions within this project.
